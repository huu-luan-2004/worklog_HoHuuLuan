## Objective

Create the main VPC for the BravelSport workshop.

## Steps

1. Open the VPC console.
2. Create a VPC named `bravelsport-vpc`.
3. Use the CIDR block `[TO BE CONFIRMED]`.
4. Enable DNS resolution and DNS hostnames.
5. Add tags such as `Project=BravelSport`, `Environment=Workshop`, and `Owner=[TO BE CONFIRMED]`.

![Create VPC](/images/5-Workshop/5.3-Network-infrastructure/create-vpc.png)

## Result Checklist

- The VPC exists in the selected Region.
- DNS support and hostnames are enabled.
- Tags are applied consistently.
- The CIDR range is recorded in the workshop parameter table.
