### Week 4 Objectives:
* Research access management solutions and practice secure connection via Remote Desktop Gateway (RD GW).
* Deploy Active Directory (AD) directory services and configure Hybrid DNS resolution between Cloud and On-Premises.
* Set up AWS CLI and establish an advanced service chain combining S3 (storage) and SNS (notifications).
* Configure advanced permissions (IAM Policies), network segmentation, launch EC2, and study Troubleshooting methods.
* Establish time-based access controls and manage Customer Managed Policies.
* Interact with IAM Identity Center Identity Store APIs and execute resource cleanup (Cleanup).
