### Week 8 Objectives:
* Study and configure AWS Storage Gateway services to bridge on-premises storage environments with AWS Cloud.
* Research and enable Data Deduplication features to optimize and save file share storage space.
* Configure and enable Shadow Copies to allow users to restore historical versions of files themselves.
* Manage user sessions, establish directory quotas, scale throughput, and resize file system capacities.
