### Week 3 Objectives:
* Learn about AWS Budgets and set up basic budgets to control actual and forecasted spending.
* Configure budget monitoring for commitment packages (Reserved Instances and Savings Plans).
* Learn and deploy metric collection and operational logging with Amazon CloudWatch.
* Setup automated alert systems (Alarms) and construct dashboards (Dashboards) to visualize infrastructure data.
* Distinguish different AWS Support plans and practice managing and creating technical Support Cases.
